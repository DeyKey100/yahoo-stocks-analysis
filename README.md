# Анализ данных об акциях Yahoo

## Описание проекта
Проект включает полный анализ исторических данных акций компании Yahoo за период с 2018 по 2023 год. Анализ выполнен с использованием Python (pandas, numpy, matplotlib, seaborn) и Yandex DataLens.

## Структура проекта
yahoo-stocks-analysis/
├── yahoo.csv # Исходные данные
├── yahoo_stocks_analysis.ipynb #Jupyter Notebook с анализом
├── README.md #Документация

## Технологии
- **Python 3.9+**
- **Pandas & NumPy** - обработка и анализ данных
- **Matplotlib & Seaborn** - визуализация
- **Yandex DataLens** - облачная аналитика
- **Jupyter Notebook** - интерактивный анализ

## Основные этапы анализа

### 1. Анализ в Python (Jupyter Notebook)
- **Загрузка и предобработка**: преобразование типов, обработка пропусков
- **Статистический анализ**: описательная статистика, корреляции
- **Визуализация**:
  - Динамика цен закрытия
  - Распределение цен
  - Корреляционная матрица
  - Boxplot по годам
- **Агрегация данных**: группировки по годам, кварталам, месяцам

### 2. Аналитика в Yandex DataLens
Ссылка: https://datalens.yandex/n8t2amzes9au7
- Создание датасета на основе CSV
- Построение чартов:
  - Линейный график динамики цен
  - Столбчатая диаграмма объемов торгов
  - Сводная таблица с ключевыми метриками
- Дашборд с фильтрами и интерактивными элементами
